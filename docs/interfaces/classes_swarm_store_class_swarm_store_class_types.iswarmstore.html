Interface ISwarmStore<P, ItemType, DbType, DBO, ConnectorBasic, PO, CO, ConnectorMain, CFO, SSO>

Implements connection to a swarm databases. After the instance will be connected to databases it allows to send request to databases connected to. Status of connection to a specific databases is available on subscription to the instance's methods.
