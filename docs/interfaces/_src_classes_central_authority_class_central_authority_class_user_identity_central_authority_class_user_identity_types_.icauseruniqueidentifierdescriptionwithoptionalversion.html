Interface ICAUserUniqueIdentifierDescriptionWithOptionalVersion

This satring allows to uniquely identify the user in the swarm. The user register on a auth provider service which gives the unique login for the user. Both of the auth provider URL and the user id on it, gives the unique identifier for the user, which can be used on the swarm as the unique identity for the user. The raw type ('object', 'string', 'number') of the serialized identity is in the constant CA_USER_IDENTITY_TYPE
