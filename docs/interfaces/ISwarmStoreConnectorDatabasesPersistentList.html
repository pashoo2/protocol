Interface ISwarmStoreConnectorDatabasesPersistentList<P, ItemType, DbType, DBO, DBL>

List of databases options opened during this session or during previous sessions (if an instanceof databasePersistantListStorage provided) and which were not dropped. If a database not opened because it's failed, then it's options won't be added to the list.
